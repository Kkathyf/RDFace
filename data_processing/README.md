# Data Processing and Synthetic Image Analysis

This module contains tools and scripts for preparing facial image data and analyzing synthetic facial images generated by various models.

---

## Folder Structure

### `preprocessing`
Custom scripts for processing raw images before training or generation. Includes:
- Resizing and normalization
- High-resolution and colorization
- landmark detection

### `synthetic_analysis`
Scripts to evaluate the realism and phenotype-consistency of generated synthetic facial images. Tasks may include:
- Embedding-based similarity analysis
- Landmark consistency checks
- Visualization of latent distributions

## External Tools (Not Included in This Repo)

The following folders point to third-party tools used for enhancing image quality and analyzing synthetic images. These are not submitted as part of this repository. Users should install or clone them separately.

| Folder | Purpose | GitHub | License |
|--------|---------|--------|---------|
| `Real-ESRGAN/` | Real-world image super-resolution | [xinntao/Real-ESRGAN](https://github.com/xinntao/Real-ESRGAN) | BSD 3-Clause |
| `DDColor/` | Exemplar-based colorization of grayscale images | [piddnad/DDColor](https://github.com/piddnad/DDColor) | Apache 2.0 |
| `RetinaFace/` | DL-based facial detector with facial landmarks | [serengil/retinaface](https://github.com/serengil/retinaface) | MIT |
| `lpips` | Learned perceptual image similarity metric | [richzhang/PerceptualSimilarity](https://github.com/richzhang/PerceptualSimilarity) | BSD 2-Clause |


> **Note:** These tools should not be redistributed. Follow the original links and licenses for use.
