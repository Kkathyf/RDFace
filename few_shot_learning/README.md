# Few-Shot Learning

This folder contains Jupyter notebooks for few-shot phenotype classification using Prototypical Networks (ProtoNet). These models are designed to generalize well from only a few training examples per class.

## Contents

| Notebook | Description |
|----------|-------------|
| `protonet_kfold.ipynb` | Baseline ProtoNet model trained and evaluated with real (non-synthetic) data using k-fold cross-validation. |
| `protonet_kfold_dream.ipynb` | ProtoNet model trained with a combination of real and DreamBooth-generated synthetic data using k-fold cross-validation. |

## Usage

1. Install required libraries (e.g., `torch`, `pandas`, `numpy`, `scikit-learn`).
2. Modify dataset paths as needed inside the notebook.
3. Run all cells to train and evaluate ProtoNet performance.

## Notes

- `dream` variant uses synthetic images generated by DreamBooth to augment the support set in each episode.
- Make sure data and label files are preprocessed and formatted to support few-shot episode creation.

---

For more information on Prototypical Networks, see the original paper:  
*Snell et al., "Prototypical Networks for Few-shot Learning", NeurIPS 2017*
